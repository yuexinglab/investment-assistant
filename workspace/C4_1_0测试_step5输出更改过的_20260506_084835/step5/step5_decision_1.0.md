# 会前初步判断

## 1. 我现在对公司的定性

**一句话判断**：东岚晶微是一家背靠集团资源的特种分子筛材料销售与工艺包项目交付公司，当前最大问题是核心价值主张（能耗/成本降低70-90%）缺乏大规模客户验证，且定制化业务模式与规模化扩张叙事之间存在根本性矛盾。

**公司本质**：材料产品销售+工艺包项目交付型公司，而非平台或技术授权公司。

**决策**：`maybe`（信心度：medium）

**核心原因**：从项目本身看，方向（工艺替代）和团队背景有吸引力，但核心数据未验证、定制化与规模化矛盾突出，存在过度包装信号。从洪总政府产业基金视角看，该项目有本地落地（建厂/子公司）和带动产业的可能，但缺乏真实客户、收入结构和产线规划等硬约束验证，当前信息不足以支持推进，需进一步深挖。


## 2. 为什么还值得继续看（优势 & 相对确定性）

1. **工艺替代（分子筛法替代精馏法）方向符合产业降本增效和国产替代趋势，市场空间大。**
   → 若数据真实，该方向对下游电子特气、高端溶剂等行业有明确经济价值，是推动项目增长的核心驱动力。

2. **背靠东岚集团，在初期资源获取、产业落地和客户导入方面有潜在优势。**
   → 对政府产业基金而言，集团背景可能加速本地落地和产业链协同，符合profile中‘工业化’和‘技术转移’偏好。

3. **项目有明确的产能建设计划（融资5000万用于产线），具备本地落地的物理基础。**
   → 符合profile中‘local_landing’硬约束，若能明确落地地点和规模，可成为推进理由。


## 3. 我最担心的几个问题（核心不确定性）

1. **核心价值主张（能耗降70-90%、成本降50-70%）缺乏大规模客户验证，数据真实性存疑。**
   → 这是整个商业逻辑的基石。若数据不实或仅限实验室，客户购买动力、营收预测和估值逻辑都将崩塌。

2. **当前业务高度定制化（项目制交付），与规模化扩张叙事存在根本矛盾，缺乏标准化复制证据。**
   → 若每个项目都需要大量定制，规模化后将面临交付瓶颈和毛利不稳定，影响估值逻辑，也与profile中‘clear_business’偏好冲突。

3. **缺乏真实客户和收入数据，无法验证商业模式是否真实运行。**
   → profile硬约束要求‘real_customer’和‘real_revenue’，当前BP未提供任何客户名称、收入结构或回款数据，不符合基金准入条件。


### 主要风险

- [P1] **能耗/成本降低数据未经第三方或客户验证，存在技术叙事与商业化现实之间的张力。**
  → 若数据不实，整个商业逻辑崩塌，客户不会因技术而采购。

- [P1] **定制化业务模式与规模化扩张之间存在矛盾，缺乏标准化复制能力。**
  → 若每个项目都需要大量定制，规模化后将面临交付瓶颈和毛利不稳定，增长不可持续。

- [P2] **团队/学术背书可能未转化为排他性壁垒，知识产权归属不明确。**
  → 若专家只是挂名或知识产权归属高校，技术壁垒脆弱，容易被竞品复制。

- [P2] **融资用途（产能扩建）与市场准入（电子特气、液相领域高壁垒）之间存在错位，存在产能闲置风险。**
  → 若客户获取进展缓慢，产能扩建将导致资金浪费和估值泡沫。
