# 投资决策框架

## 核心判断

**一句话判断**：本质是硬件制造商/项目交付型公司，当前最大问题是BP用平台化/软件/全球化高估值标签包装一个本质为硬件销售和项目交付的业务，导致收入结构与未来叙事严重错位。

**公司本质**：硬件制造商/项目交付型公司

**决策**：`maybe`（信心度：medium）

**核心原因**：公司已进入头部主机厂供应链，2024年发货超5000万，商业化初步验证；但核心壁垒（国产芯片可靠性、AI作用）和增长可持续性（政策驱动vs经济ROI）尚未验证，估值逻辑存在错位风险。


## 值得继续看

1. **已切入中国重汽等头部主机厂供应链，2024年发货金额超5000万，表明产品通过初步验证，商业化处于爆发期。**
   → 进入头部主机厂是商业化的重要里程碑，证明产品有真实客户买单，降低技术风险。

2. **xPower900集成方案能为整车厂直接降低2000-3000元成本，在成本极度敏感的商用车市场具有显著竞争力。**
   → 成本优势是驱动客户切换供应商的核心动力，若能验证可持续性，则增长逻辑成立。

3. **模块化平台（硬件复用率70%+，软件复用率80%+）和全面集成国产芯片构成潜在技术壁垒，若验证成功，可形成差异化优势。**
   → 如果国产芯片的长期可靠性和成本优势被验证，公司可能建立护城河，支撑高估值。


## 为什么不投

1. **当前收入几乎全部来自硬件销售和项目交付，BP却用平台化、软件、全球化高估值标签包装，导致估值逻辑错位。**
   → 如果公司本质是硬件制造商，当前估值倍数可能过高，投资回报预期不匹配。

2. **全面集成国产芯片的长期可靠性尚未验证，且集成BDU多合一方案并非独创（汇川技术、比亚迪已有类似方案），技术壁垒可能被夸大。**
   → 如果国产芯片可靠性风险暴露，或技术领先性被证伪，公司护城河将大幅削弱，面临同质化竞争。

3. **工控业务和全球化战略缺乏实质性收入和数据支撑，第二曲线可能只是叙事，当前增长高度依赖单一业务（商用车电控）。**
   → 如果单一业务增长不及预期，或客户集中度高导致风险暴露，公司缺乏第二增长点支撑估值。


## 核心风险

- [P1] **国产芯片在商用车极端工况下的长期可靠性未验证，可能被客户拒绝或导致高故障率。**
  → 这是公司核心壁垒的基石，若可靠性不达标，成本优势将被抵消，客户可能转向更可靠的方案。

- [P1] **收入结构不透明，BP未披露硬件、项目、软件/服务收入占比，可能掩盖低毛利硬件制造的本质。**
  → 如果毛利率低于20%，公司本质是低毛利硬件制造商，当前估值逻辑不成立。

- [P2] **客户需求可能主要受政策驱动（新能源补贴、路权政策），政策退坡后增长可持续性存疑。**
  → 如果客户采购不是基于经济ROI，政策退坡后收入可能大幅下滑，投资风险高。


## 必问问题

1. 如果去掉所有关于平台化、软件、全球化的叙事，只算你们卖硬件和做项目的收入，2024年的毛利率是多少？这个数字能支撑你们现在的估值吗？
   目的：验证收入结构真相，判断公司本质是硬件制造商还是科技平台。

2. 如果现在把你们所有的AI和国产芯片能力都拿掉，xPower900还能不能正常生产、正常卖给中国重汽？如果不能，具体缺了哪一环？如果能，那你们的核心壁垒到底是什么？
   目的：验证技术壁垒虚实，判断AI/国产芯片是核心壁垒还是包装叙事。

3. 如果现在新能源商用车补贴全部取消，路权政策也恢复到和燃油车一样，你觉得2025年还能实现1.6亿的收入预测吗？如果不能，你估计会降到多少？
   目的：验证增长可持续性，判断客户需求是政策驱动还是经济ROI驱动。

**次要逻辑**：硬科技商业化型, 政策驱动型

**风险类型**：技术可靠性风险, 收入结构不透明风险, 政策依赖风险
