# 投资决策框架

## 核心判断

**一句话判断**：这是一家以自动轮（轮毂电机+线控制动+线控转向三合一）为核心技术的底盘Tier 1公司，当前最大问题是技术工程化验证缺失和商业化路径不清晰，从实验室样机到车规级量产的鸿沟尚未跨越。

**公司本质**：硬科技商业化型公司，本质是底盘域核心Tier 1，技术专利驱动，但当前更接近技术开发/项目制交付型，而非BP暗示的制造/产品销售型。

**决策**：`maybe`（信心度：medium）

**核心原因**：技术方向正确且特种车辆场景价值明确，但工程化验证、客户合作深度和团队产业化能力三大关键缺口均未闭合，需进一步尽调才能判断是否值得推进。


## 值得继续看

1. **自动轮三合一集成方案（轮毂电机+线控制动+线控转向）在行业内具有独特性，专利布局构成技术壁垒，且行业趋势（线控底盘、滑板底盘）明确。**
   → 如果技术壁垒成立，公司有望在底盘域形成差异化竞争优势，成为核心Tier 1供应商，具备高估值潜力。

2. **特种车辆（军工、物流）对极致机动性有明确需求，且对成本敏感度较低，BP展示的‘横行’、‘原地掉头’等功能在此场景下有直接价值。**
   → 特种车辆场景可提供早期收入和市场验证窗口，降低商业化风险，是公司从0到1的关键切入点。

3. **公司已获得行业奖项（AITC2024最具投资价值项目）和初步客户合作（北汽、华为等），表明技术方案在专业领域内获得了一定认可。**
   → 行业认可和初步客户合作是硬科技公司商业化的常见早期信号，降低了纯学术项目的风险，为后续尽调提供了基础。


## 为什么不投

1. **轮毂电机方案固有的工程难题（簧下质量、散热、耐久性）是行业共识，BP未提供任何车规级耐久性测试数据（DV/PV），技术可行性存疑。**
   → 如果基础工程问题未解决，所有技术优势（减重、降本）都是空中楼阁，项目可能停留在实验室阶段，无法量产。

2. **BP列举的客户（北汽、华为等）合作模式不清晰，缺乏量产定点合同（SOP）和收入数据，商业化叙事可能严重包装。**
   → 如果客户合作仅为技术开发或样件阶段，公司当前估值缺乏支撑，且从技术服务向重资产制造转型的风险极高。

3. **团队以学术背景为主，缺乏具备大规模汽车零部件制造、质量管理、供应链管理经验的产业界核心成员，产业化落地能力存疑。**
   → 从教授创业到产业公司，需要产业经验丰富的核心成员主导工程化、量产和供应链管理，否则量产计划可能无法执行。


## 核心风险

- [P1] **技术工程化风险：自动轮模块未通过车规级耐久性测试（DV/PV），簧下质量、散热、耐久性等工程难题未解决，技术优势可能无法量产。**
  → 这是判断技术是否具备量产可行性的核心依据，如果未验证，所有技术壁垒和成本优势都是理论假设。

- [P1] **商业化叙事包装风险：客户合作（北汽、华为等）可能仅为技术开发或样件阶段，缺乏量产定点合同和收入数据，商业化进展远低于BP暗示。**
  → 客户合作深度直接决定公司当前估值是否合理，如果只是意向合作，公司本质更接近技术开发公司，而非底盘系统供应商。

- [P2] **团队产业化能力风险：核心团队以学术背景为主，缺乏汽车零部件量产经验，从实验室到量产的转化能力存疑。**
  → 学术成果转化为可靠产品需要工程化、品控、成本控制等能力，如果团队无产业经验，量产计划可能无法执行。


## 必问问题

1. 靳教授，如果现在有一家主机厂要求你们提供自动轮模块的DV/PV测试报告，才能签量产定点合同，你们能提供吗？如果不能，那BP里提到的北汽、华为这些客户，他们是在什么阶段和你们合作的？
   目的：验证技术工程化验证的真实性，排除技术包装风险。

2. 靳教授，如果现在北汽或华为要求你们提供自动轮模块的量产定点合同，你们能立刻签吗？如果不能，那BP里把他们列为客户，是不是更多是‘意向客户’或者‘潜在客户’，而不是已经产生收入的‘实际客户’？
   目的：验证客户合作的真实性和深度，排除商业化叙事包装风险。

3. 靳教授，如果现在有一家主机厂要求你们在18个月内交付1000套自动轮模块，你们团队里谁能确保按时、按质、按成本交付？如果没有人能，那这轮融资是不是应该优先用来引进一位有量产经验的产业合伙人？
   目的：评估团队产业化能力，判断从学术到产业的转化风险。


## 投资逻辑归因

**核心逻辑**：待验证：硬科技商业化型 vs 技术开发/项目制交付型

**次要逻辑**：数据/AI模型驱动型（远期）, 政策驱动型（风险）

**风险类型**：技术工程化风险, 商业化叙事包装风险, 团队产业化能力风险
