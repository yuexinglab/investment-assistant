# 投资决策框架

## 核心判断

**一句话判断**：斯年智驾本质是重资产、项目制的物流装备制造与运营公司，而非AI科技平台公司；当前最大问题是代运营模式的真实财务模型（资产归属、毛利率、回本周期）未披露，导致估值逻辑混乱。

**公司本质**：重资产运营服务商（代运营模式需自持车辆资产）与项目制交付型系统集成商（非标项目定制）的混合体，AI/数据能力为内部效率工具，非核心收入来源。

**决策**：`maybe`（信心度：medium）

**核心原因**：方向（港口无人驾驶）靠谱，代运营模式差异化，但核心矛盾（轻资产叙事 vs 重资产现实）未解决，需验证代运营财务模型和客户真实付费驱动力后才能决定是否推进。


## 值得继续看

1. **代运营模式降低客户决策门槛，形成持续收入流，且已部署240台车（行业第一），验证了场景落地能力。**
   → 如果代运营模式财务模型健康（毛利率>30%，回本周期<3年），则公司可视为有壁垒的运营服务商，增长路径清晰。

2. **港口无人驾驶客户痛点明确（降本增效、安全合规），且BP中客户价值逻辑成立（人工和折旧占成本60%+，无人驾驶可降低80%事故率）。**
   → 如果大部分客户是纯商业ROI驱动（非政策补贴），则收入可持续性强，市场空间真实。

3. **全栈自研与场景数据积累（上千万公里、2年无间断运营）构成真实壁垒，数据飞轮效应在封闭场景中难以短期复制。**
   → 如果国产芯片替代方案（地平线/黑芝麻）已测试通过且AI大模型有量化提效数据，则技术壁垒可靠，可支撑长期竞争力。


## 为什么不投

1. **轻资产科技公司叙事与重资产制造运营现实存在根本矛盾：BP以AI/平台公司定位融资，但代运营模式需公司自持车辆资产，导致估值逻辑混乱（科技公司估值 vs 重资产运营公司回报率）。**
   → 若代运营模式需公司自持资产且毛利率<20%、回本周期>5年，则公司本质是重资产运营公司，增长依赖持续融资，投资人回报路径不清晰。

2. **标准化产品故事与项目制交付现实矛盾：BP强调快速部署，但每个港口/园区场景差异大，部署周期从2个月到8个月不等，规模化复制能力存疑。**
   → 如果项目非标程度高，增长将受限于交付团队规模和客户定制需求，毛利率可能被侵蚀，不适合按产品公司估值。

3. **客户需求可能混合政策驱动因素：BP未区分客户类型，中国港口无人驾驶推广初期大量项目由政府补贴或国企示范任务驱动，纯商业场景付费意愿待验证。**
   → 若超过60%客户依赖政策补贴，政策退坡后收入可持续性存疑，市场空间有限。


## 核心风险

- [P1] **代运营模式重资产风险：公司需自持车辆资产，每扩张100台车可能需数亿元资金，BP未披露资产负债率和现金流状况，规模化后资金链可能断裂。**
  → 这是Step3B核心矛盾（轻资产叙事 vs 重资产现实）的直接体现，决定公司是苦生意还是可持续商业模式。

- [P1] **客户集中度与政策依赖风险：客户以国有港口集团为主（如宁波港、天津港、青岛港），前五大客户收入占比可能较高，且部分客户可能为政策示范项目驱动。**
  → Step3B tension指出客户价值可能混合政策因素，若政策退坡或单一客户流失，收入将大幅下降。

- [P2] **供应链依赖风险：核心域控使用英伟达芯片，国产替代方案（地平线/黑芝麻）尚在开发中，2026年量产计划未验证，存在被卡脖子风险。**
  → Step3B overpackaging signal指出技术领先性缺乏量化对比数据，若芯片被限制，技术壁垒将不复存在。

- [P2] **AI能力包装风险：TransportGPT大模型缺乏独立第三方验证或量化提效数据，可能只是内部辅助工具而非核心壁垒。**
  → Step3B overpackaging signal指出过度强调AI/大模型，但核心产品仍依赖传统算法，若AI能力被证伪，融资叙事将崩塌。


## 必问问题

1. BP提到代运营模式是行业内首次提出且业绩领先，能具体说说目前240台车中，代运营模式占多少台？这些代运营车辆，资产是归客户所有，还是公司自己购买持有？折旧和维修成本由谁承担？
   目的：验证代运营模式的资产归属和财务模型，判断公司是轻资产运营还是重资产自持。

2. 以一台代运营车辆为例，年收入大概多少？扣除折旧、人工、电费、维护后，毛利率大概在什么范围？
   目的：验证代运营模式的单台车盈利能力和回本周期，判断商业可持续性。

3. BP提到覆盖20+客户，单体部署规模超50台。能具体说说这些客户中，有多少是纯商业ROI驱动（没有政策补贴或示范要求）？这些客户中，有多少是从试点项目转向规模化采购的？续约率大概多少？
   目的：验证客户付费的真实驱动力，判断收入可持续性。

4. BP提到核心域控使用英伟达芯片，同时国产替代方案（地平线/黑芝麻）在开发中。目前替代方案的测试进展如何？性能达到英伟达Orin的多少？
   目的：评估国产芯片替代方案的技术成熟度，判断供应链风险。

5. BP中提到的TransportGPT大模型，具体在哪些环节提升了效率？有没有量化数据，比如减少实验次数或提升效率百分比？如果没有TransportGPT，你们现有的无人重载车辆还能稳定运营吗？
   目的：验证AI大模型是核心能力还是融资叙事，判断技术壁垒的真实性。


## 投资逻辑归因

**核心逻辑**：待验证：重资产运营型 vs 项目制交付型

**次要逻辑**：制造/产品销售型, 数据/AI模型驱动型（内部工具）

**风险类型**：重资产运营现金流风险, 非标项目交付风险, 政策驱动需求风险, 供应链依赖风险
