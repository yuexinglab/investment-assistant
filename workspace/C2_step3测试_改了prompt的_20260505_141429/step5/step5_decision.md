# 投资决策框架

## 核心判断

**一句话判断**：这是一家以实验室稳定性世界纪录为核心叙事、但尚未验证量产工程化能力的钙钛矿模组公司，当前最大问题是实验室数据与量产能力之间的鸿沟未被弥合，且零收入现实与多产品线扩张故事严重错位。

**公司本质**：待验证：硬科技商业化型 vs 项目制交付型

**决策**：`maybe`（信心度：medium）

**核心原因**：技术壁垒（稳定性）方向正确且稀缺，但工程化能力和商业化路径均未验证，需通过关键缺口验证后才能决定是否推进。


## 值得继续看

1. **稳定性世界纪录（50000小时）是钙钛矿产业化的核心痛点，若能在模组级别复现，将构成显著技术壁垒。**
   → 这是公司差异化定位的基础，也是潜在护城河，值得投入资源验证其真实性。

2. **公司已建成20MW中试线，而非仅停留在实验室阶段，具备初步工程化尝试。**
   → 中试线是跨越‘死亡之谷’的必要步骤，如果运行数据良好，可大幅降低量产风险。

3. **与头部车企、玻璃企业的战略合作表明公司具备一定的产业资源对接能力，若转化为真实订单，可加速商业化。**
   → 头部客户的背书和潜在订单是验证市场需求和商业可行性的关键信号。


## 为什么不投

1. **实验室稳定性数据（小面积器件）与量产模组性能之间存在巨大鸿沟，BP完全回避了模组级别数据和工程化瓶颈。**
   → 如果模组级别无法复现实验室数据，技术壁垒的含金量将大打折扣，投资逻辑不成立。

2. **当前零收入，但BP规划了6条产品线（电站、BIPV、卫星、智能项圈、室内光伏、AI平台），收入预测（2025年1500万）无任何客户订单支撑。**
   → 多产品线扩张故事与零收入现实严重错位，投资人无法判断真实增长点，存在估值泡沫风险。

3. **团队以学术背景为主，缺乏光伏产业大规模量产、供应链管理、市场销售等关键商业经验，且BP未展示任何补强计划。**
   → 钙钛矿产业化的核心挑战是工程化量产能力，而非实验室技术。团队能力短板可能导致无法跨越‘死亡之谷’。


## 核心风险

- [P1] **实验室稳定性世界纪录（50000小时）在模组级别（300mm*300mm）无法复现，或中试线良率低于60%。**
  → 这是公司核心壁垒的基石，若无法复现，技术优势消失，公司沦为普通钙钛矿初创企业。

- [P1] **与头部车企、玻璃企业的合作仅为框架协议或意向，无任何收入或明确订单。**
  → 当前零收入，若合作无实质进展，2025年收入预测完全落空，公司估值缺乏支撑。

- [P1] **团队无产业经验补强计划，认为现有学术团队足以应对量产工程化挑战。**
  → 钙钛矿量产涉及涂布、激光刻划、封装等复杂工艺，缺乏产业经验将导致良率、成本失控。

- [P2] **多产品线扩张分散资源，导致所有方向都无法突破，公司无法聚焦核心市场。**
  → 在零收入阶段同时推进6条产品线，每条都需要不同的渠道、认证和客户，极可能导致资源浪费和战略失焦。


## 必问问题

1. 你们说50000小时稳定性是世界纪录，但这是小面积器件的数据。如果现在让第三方机构对你们中试线生产的300mm*300mm模组进行同样的稳定性测试，你们有信心复现这个数据吗？如果不行，差距有多大？
   目的：验证实验室稳定性数据在量产层面的真实价值，判断技术壁垒的含金量。

2. 你们和全球玻璃龙头、头部新能源车企的合作，到底是已经签了采购合同、有明确付款条款，还是只是签了战略合作协议、没有任何商业约束？如果只是意向，你们2025年1500万的收入预测，有多少是来自这些意向订单？
   目的：验证合作是真实订单还是框架协议，判断商业化真实进展和收入预测的可靠性。

3. 你们团队里，有没有任何一个人，曾经成功把一项实验室技术转化为量产产品，并且在这个过程中负责过工艺、设备、供应链或质量管理？如果没有，你们凭什么认为自己能跨越从实验室到量产的‘死亡之谷’？
   目的：评估团队是否具备将技术商业化的工程化能力，判断量产风险。
