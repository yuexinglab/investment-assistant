# 投资决策框架

## 核心判断

**一句话判断**：这是一家以集团资源为依托、当前依赖非标项目交付的分子筛解决方案提供商，其核心矛盾在于用‘硬科技+国产替代’的高估值叙事包装一个缺乏收入验证和标准化能力的项目制公司。

**公司本质**：项目制交付型材料解决方案提供商，非标准化产品公司或平台公司

**决策**：`maybe`（信心度：medium）

**核心原因**：方向靠谱（工艺替代降本增效显著），但当前业务模式（非标项目交付）与未来故事（标准化产品规模化）之间存在巨大张力，且技术壁垒和营收预测均缺乏商业化数据支撑。


## 值得继续看

1. **工艺替代（分子筛替代精馏）的降本增效数据（能耗降70-90%、成本降50%）具有颠覆性，若在客户产线得到验证，将构成强壁垒。**
   → 这是公司最核心的卖点和增长假设，若验证为真，则公司具备从项目制向标准化产品转型的基础，可支持高估值。

2. **背靠东岚集团，在市场、人才、资金和产业落地经验上有显著优势，降低了早期风险。**
   → 集团资源可帮助公司更快获得客户验证和产能建设支持，缩短商业化周期。

3. **电子特气、高纯溶剂、氢能等下游赛道均为高增长、高壁垒、国产替代需求迫切的领域，市场空间大。**
   → 即使公司当前是项目制，只要能在某个细分赛道建立标杆客户和复购，仍可逐步标准化，具备长期价值。


## 为什么不投

1. **当前业务高度依赖非标项目交付，BP未提供任何客户复购率、项目模块复用率或交付周期数据，规模化故事缺乏基础。**
   → 若每个项目都需要从头定制，则营收预测（2026-2028年从5000万到1.5亿）和产能扩建计划（融资5000万）缺乏支撑，投资风险高。

2. **技术壁垒（独创工艺、能耗下降70-90%）未得到客户产线验证或第三方测试，BP中仅使用模糊技术术语，缺乏量化指标。**
   → 若技术优势仅停留在实验室或PPT阶段，则公司本质是普通材料供应商，不应按硬科技公司估值。

3. **营收预测和融资用途远大于当前业务描述，缺乏历史数据支撑，存在估值泡沫风险。**
   → 若融资仅基于市场预测而非已锁定的客户需求，则资金可能被低效使用，且收入不达预期时公司可能陷入困境。


## 核心风险

- [P1] **当前业务（项目制交付/定制化分子筛销售）与未来故事（标准化产品规模化、高增长营收预测）之间的张力未解决，若无法标准化，则增长不可持续。**
  → 这是公司估值逻辑的核心矛盾：若当前是项目制，则应按项目公司估值（低倍数）；若未来能标准化，才可按产品公司估值（高倍数）。

- [P1] **技术壁垒（能耗下降70-90%）缺乏客户产线验证，若仅基于实验室模拟，则技术优势可能被竞争对手快速复制。**
  → 技术壁垒是公司获得高估值溢价的核心假设，若无法验证，则估值逻辑崩塌。

- [P1] **营收预测（2026-2028年从5000万到1.5亿）和融资用途（产能扩建）缺乏历史数据支撑，存在估值泡沫风险。**
  → 若融资仅基于市场预测而非已锁定的客户需求，则资金可能被低效使用，且收入不达预期时公司可能陷入困境。

- [P2] **科学家团队（如杨维X、李秀X）可能仅为顾问或挂名，而非全职核心研发人员，技术商业化落地能力存疑。**
  → 若科学家团队未深度参与产品化，则公司实际技术能力可能远低于BP叙事。
