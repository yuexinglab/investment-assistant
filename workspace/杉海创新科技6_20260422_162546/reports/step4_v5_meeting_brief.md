# 📌 本场会议目标

一句话判断：验证杉海创新的“AI平台”到底是不是核心引擎，“客户粘性”是不是真依赖，“跨行业扩张”是不是有真金白银。

# 🎯 优先搞清楚的3件事

1. **AI平台“建沐™”的真实效用**：是核心研发引擎，还是辅助工具？——决定估值逻辑是平台型还是配方型。
2. **客户付费的真实动机与粘性**：是技术依赖，还是短期卖点？——决定收入可持续性和定价权。
3. **跨行业扩张（食品）的真实商业化进展**：是真实收入，还是故事？——决定“平台化”叙事是否成立。

# 🧭 建议提问路径

先问业务模式（从具体产品切入拆解AI） → 再问客户合作细节（判断粘性深度） → 最后问食品业务订单数据（验证跨行业真实性）。全程保持质疑节奏，用trap问题揭示矛盾。

# 🧩 主题1：AI平台“建沐™”的真实效用

**想搞清楚：** AI在具体产品研发中到底起了多大作用？有没有量化证据？

**主问题（来自 opening）：**
“我们注意到你们最成功的产品，比如超分子VC，从概念到量产大概花了多久？”

**理想信号：**
- 能清晰描述AI参与的具体环节（如虚拟筛选、性能预测）
- 能给出量化数据（如“筛选了1000种组合，AI预测准确率80%”）
- 承认AI是核心流程的一部分，而非可有可无

**危险信号：**
- 回答模糊，如“AI帮助我们提高了效率”
- 无法提供任何量化对比数据
- 暗示没有AI也能做出来，只是慢一点

**如果对方回答偏正面（来自 deepen）：**
“在这个研发过程中，‘建沐™’AI平台具体参与了哪些环节？是帮你们筛选了共晶组合，还是预测了稳定性？”

**如果对方回答模糊（来自 trap）：**
“如果当时没有这个AI平台，你们觉得这个产品还能做出来吗？大概会慢多久？”

# 🧩 主题2：客户付费的真实动机与粘性

**想搞清楚：** 客户为什么选杉海创新？是技术依赖，还是短期卖点？

**主问题（来自 opening）：**
“你们和欧莱雅、宝洁的合作，具体是哪种模式？是联合研发，还是直接采购你们的原料？”

**理想信号：**
- 原料用于客户的核心大单品
- 有独家或长期供应协议
- 客户主动提出联合研发需求

**危险信号：**
- 仅用于新系列或限量版
- 无独家协议，客户有多个供应商
- 合作模式为一次性项目

**如果对方回答偏正面（来自 deepen）：**
“他们采购的原料，是用在哪些具体产品上？是限量版、新系列，还是像Olay、兰蔻这样的核心大单品？”

**如果对方回答模糊（来自 trap）：**
“这些客户有没有跟你们签独家供应协议？或者，他们有没有同时也在测试其他供应商的类似产品？”

# 🧩 主题3：跨行业扩张（食品）的真实商业化进展

**想搞清楚：** 食品业务到底有多少真金白银？是故事还是现实？

**主问题（来自 opening）：**
“BP提到食品原料订单年增长率达300%，这个增长是从一个多大的基数开始的？比如2023年食品业务的收入大概是多少？”

**理想信号：**
- 能给出具体的收入基数（如“2023年100万，2024年400万”）
- 订单来自知名企业，且已进入批量采购
- 有明确的复购率数据（如“复购率>50%”）

**危险信号：**
- 回避基数问题，只谈增长率
- 订单多为意向或测试订单
- 无复购或复购率极低

**如果对方回答偏正面（来自 deepen）：**
“这些订单主要是跟安利、百事这些大客户的，还是来自一些中小客户？其中有多少是已经交付并收到钱的，而不是意向订单？”

**如果对方回答模糊（来自 trap）：**
“这些客户在拿到你们的样品或首批产品后，有没有下第二单、第三单？复购率大概是多少？”

# 📊 最终判断标准

**如果AI在至少一个核心产品（如超分子VC）的研发中，提供了明确的、可量化的预测和加速作用，且该作用被研发团队认为是关键成功因素；同时至少有一家头部客户将杉海创新的原料用于其核心产品，并签订了长期（如3年以上）或独家供应协议；且食品业务2024年真实收入超过500万元，至少有一个知名客户已进入批量采购阶段** → 那么“AI+材料平台”的估值逻辑成立，值得高估值。

**如果AI的作用被描述为“辅助”、“加速文献调研”，或无法提供任何量化数据；所有客户合作均为短期项目制，无独家或长期协议；食品业务收入低于100万元，或所有客户均处于研发/测试阶段** → 那么公司本质上是一家配方优化公司，估值逻辑应回归到高端配方型，平台溢价不成立。
