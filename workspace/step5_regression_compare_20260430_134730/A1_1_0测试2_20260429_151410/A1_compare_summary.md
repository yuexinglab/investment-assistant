## 4. Must Ask Questions 内容对比

1. **何总，BP中提到代运营是公司核心模式，而且业绩领先。我们想了解一下，一个典型的代运营项目，比如在宁波港部署50台车，大概的投资额、年收入和回本周期是怎样的？**
   - 验证目的：验证代运营模式的真实财务模型，判断其可持续性和规模化路径。
2. **何总，BP中提到TransportGPT和AI大模型，这些AI能力现在主要用在哪些具体环节？如果现在把TransportGPT关掉，只用传统算法，你们的无人重载车辆还能稳定运营吗？效率会下降多少？**
   - 验证目的：验证AI/数据能力是否构成核心壁垒，还是只是研发/交付效率工具。
3. **何总，BP中提到客户主要是港口和园区，这些客户采购你们的无人驾驶方案，主要是为了降本增效，还是因为政策要求？如果政府明天取消所有港口无人化补贴，你觉得现有客户中，有多少还会继续用你们的方案？**
   - 验证目的：验证客户需求是政策驱动还是真实经济ROI驱动，判断收入的抗政策风险能力。


**新版**（共 5 个）：
1. **何总，BP中提到代运营是公司核心模式，而且业绩领先。我们想了解一下，一个典型的代运营项目，比如在宁波港部署50台车，大概的投资额、年收入和回本周期是怎样的？如果客户合同只签了2年，到期后不续约，这车怎么办？折旧成本谁来承担？**
   - 验证目的：验证代运营模式的真实财务模型和盈利性，判断公司商业模式的资产轻重和可持续性。
2. **何总，BP中提到TransportGPT和AI大模型，这些AI能力现在主要用在哪些具体环节？如果现在把TransportGPT关掉，只用传统算法，你们的无人重载车辆还能稳定运营吗？效率会下降多少？**
   - 验证目的：验证AI/数据能力是否构成核心壁垒，判断公司技术壁垒的虚实和估值逻辑的合理性。
3. **何总，BP中提到客户主要是港口和园区，这些客户采购你们的无人驾驶方案，主要是为了降本增效，还是因为政策要求？如果政府明天取消所有港口无人化补贴，你觉得现有客户中，有多少还会继续用你们的方案？能给出一个具体的百分比吗？**
   - 验证目的：验证客户需求是政策驱动还是商业ROI驱动，判断公司收入的抗政策风险能力和长期增长逻辑。
4. **何总，BP中强调增材制造降本80%，这个数据是基于年产多少台的测算？当前良品率、生产节拍、材料成本具体是多少？如果大规模量产（比如年产5000台），成本优势还能保持吗？**
   - 验证目的：验证增材制造降本数据的真实性和可规模化性，判断公司成本控制能力和供应链壁垒。
5. **何总，你们去年收入大概多少？今年预计能到多少？这个增长主要来自老客户复购加车，还是新场景（如散杂货、工厂基地）的拓展？前五大客户集中度如何？**
   - 验证目的：验证公司收入增长的真实驱动力和客户集中度风险，判断公司业务的稳定性和可复制性。
