## 4. Must Ask Questions 内容对比

**旧版**（共 3 个）：
1. **靳教授，如果现在有一家主机厂要求你们提供自动轮模块的DV/PV测试报告，才能签量产定点合同，你们能提供吗？如果不能，那BP里提到的北汽、华为这些客户，他们是在什么阶段和你们合作的？**
   - 验证目的：验证技术工程化验证的真实性，排除技术包装风险。
2. **靳教授，如果现在北汽或华为要求你们提供自动轮模块的量产定点合同，你们能立刻签吗？如果不能，那BP里把他们列为客户，是不是更多是‘意向客户’或者‘潜在客户’，而不是已经产生收入的‘实际客户’？**
   - 验证目的：验证客户合作的真实性和深度，排除商业化叙事包装风险。
3. **靳教授，如果现在有一家主机厂要求你们在18个月内交付1000套自动轮模块，你们团队里谁能确保按时、按质、按成本交付？如果没有人能，那这轮融资是不是应该优先用来引进一位有量产经验的产业合伙人？**
   - 验证目的：评估团队产业化能力，判断从学术到产业的转化风险。


**新版**（共 3 个）：
1. **靳教授，我们看到BP里提到自动轮模块能减重30%、降本30%，这些数字很吸引人。想了解一下，目前自动轮模块有没有进行过车规级的耐久性测试？比如高低温循环、涉水、振动这些？有没有具体的测试报告或者关键数据可以分享一下？**
   - 验证目的：验证技术从实验室样机到车规级量产的可行性，排除工程化风险。如果已通过DV/PV测试且数据达标，则技术壁垒成立；否则应直接否决或要求极低估值进入。
2. **我们看到BP里提到和北汽、华为、广汽这些客户有合作，想了解一下，这些合作目前具体是什么模式？是技术开发、样件供货，还是已经有量产定点合同（SOP）了？有没有产生收入？**
   - 验证目的：验证客户合作的真实性和深度，排除商业化叙事包装。如果存在量产定点合同，则商业化路径清晰；如果只是技术开发或样件阶段，则估值应大幅下调。
3. **靳教授，我们很认可您在吉林大学的技术积累。想了解一下，目前公司的核心团队除了您，还有哪些成员？比如负责工程化、生产制造、供应链这些环节的负责人，他们之前是在哪些公司工作的？有没有在博世、大陆、采埃孚这些Tier 1做过量产项目的经验？**
   - 验证目的：评估团队是否具备从学术到产业的转化能力，排除团队/背书叙事型风险。如果团队中缺乏产业界量产经验的核心成员，则产业化落地能力存疑，应要求引入产业合伙人或降低估值。
