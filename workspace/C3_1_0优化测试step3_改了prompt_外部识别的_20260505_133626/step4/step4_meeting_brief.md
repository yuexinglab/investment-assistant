# 最终判断标准

**投的逻辑**：
- 公司能清晰拆分2024年收入中，来自xPower系列（商用车）和iFAST系列（工控）的占比，并能提供至少一个持续收费的客户案例（如运维合同、订阅服务）。
- 公司能提供至少一个客户（如中国重汽）对国产芯片方案的认可证明，并能说明核心芯片（如SiC IGBT）的供应商和备选方案。
- 公司能提供至少两个不同场景（如重卡和工程机械）的项目案例，并说明其中复用模块的比例（硬件70%+，软件80%+）。

**不投的逻辑**：
- 公司无法拆分收入结构，或承认当前收入几乎全部来自一次性硬件销售，且无任何持续收费模式。
- 公司无法提供任何第三方验证或客户背书，或承认国产芯片在可靠性/成本上并无显著优势。
- 公司承认每个项目都需要大量定制，或无法提供复用率数据。
