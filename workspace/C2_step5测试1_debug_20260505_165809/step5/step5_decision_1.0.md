# 投资决策框架

## 核心判断

**一句话判断**：黑冕光电是一家以实验室稳定性世界纪录为核心卖点、尚未验证量产能力和商业收入的早期钙钛矿技术公司，当前最大问题是实验室优势能否转化为量产竞争力以及是否存在真实收入。

**公司本质**：技术解决方案公司（中试阶段、项目制交付），而非产品公司

**决策**：`maybe`（信心度：medium）

**核心原因**：技术壁垒（稳定性世界纪录）真实且稀缺，但量产复现性、收入验证和资源聚焦均存在重大缺口，需进一步深挖才能判断是否值得推进。


## 值得继续看

1. **稳定性世界纪录（50000小时）是钙钛矿行业最稀缺的技术指标，若能在中试模组上复现，将构成显著竞争壁垒。**
   → 这是公司核心差异化优势，也是估值支撑点。若验证通过，可大幅提升投资价值。

2. **公司选择利基市场（智能穿戴、BIPV、畜牧）作为切入点，避开与晶硅正面成本竞争，逻辑上可行。**
   → 若这些市场已有真实客户订单或验证案例，则商业化路径清晰，风险可控。


## 为什么不投

1. **实验室稳定性数据（小面积器件）与中试模组性能之间存在巨大鸿沟，BP未提供任何大面积模组的第三方认证数据。**
   → 若无法复现，技术壁垒大幅缩水，公司估值逻辑崩塌。

2. **公司无任何确认收入，所有产品线均处于意向订单或战略合作阶段，商业化进展严重滞后于BP叙事。**
   → 估值完全基于技术叙事，缺乏收入验证，风险极高。

3. **多产品线叙事（光伏、BIPV、智能项圈、AI材料）与15人团队、单一中试线现实严重不匹配，资源分散风险高。**
   → 若无法聚焦，可能导致所有产品线均无法突破，失败概率大增。


## 核心风险

- [P1] **实验室稳定性世界纪录（50000小时）在300mm*300mm中试模组上无法复现，或复现后性能大幅衰减。**
  → 这是公司核心壁垒，若无法工程化，技术优势消失，估值逻辑不成立。

- [P1] **公司当前无任何真实收入，所有收入规划（2026年5000万元）缺乏现实基础。**
  → 无收入意味着商业模式未验证，估值完全依赖叙事，风险极高。

- [P2] **多产品线叙事导致资源分散，团队无法聚焦突破任何一个市场。**
  → 15人团队同时推进5个以上产品线，失败概率高，且掩盖了交付能力不足的风险。


## 必问问题

1. 你们在Science上发表的50000小时稳定性记录，是在什么尺寸的器件上测得的？
   目的：验证实验室稳定性数据是否来自小面积器件，判断技术壁垒的真实性。

2. 目前300mm*300mm模组的效率是19.6%，这个模组有没有做过同等标准的稳定性测试？结果如何？
   目的：验证实验室优势能否在量产模组上复现，判断工程化能力。

3. 如果现在有一家头部电站投资方要求你们提供300mm模组的第三方稳定性认证报告，否则不签意向订单，你们能提供吗？
   目的：测试公司是否具备量产级认证能力，识别包装信号。

4. 公司成立到现在，有没有产生过任何收入？
   目的：确认当前商业化进展，判断收入规划是否真实。

5. 如果现在要求看公司银行流水和销售合同，你能提供吗？
   目的：验证收入真实性，识别是否仅有意向订单。

6. 如果只能选一个产品线先做，你们会选哪个？为什么？
   目的：评估资源聚焦能力，判断多产品线叙事是否掩盖了战略混乱。


## 投资逻辑归因

**核心逻辑**：待验证：硬科技商业化型 vs 项目制交付型

**次要逻辑**：制造/产品销售型, 数据/AI模型驱动型

**风险类型**：硬科技商业化风险, 非标项目交付风险, 融资估值叙事风险
