# 投资决策框架

## 核心判断

**一句话判断**：这是一家以实验室稳定性记录为核心卖点、但尚未验证量产转化能力的钙钛矿组件初创公司，当前最大问题是实验室技术领先与产业化落地能力之间的巨大鸿沟。

**公司本质**：技术型公司，而非量产型公司

**决策**：`maybe`（信心度：medium）

**核心原因**：技术方向正确且稳定性记录是行业痛点，但量产验证、客户订单和团队产业化能力均未证实，存在重大不确定性。


## 值得继续看

1. **稳定性记录直击钙钛矿行业最大痛点，若能在量产中复现，将构成显著技术壁垒。**
   → 这是公司最核心的差异化优势，也是BP中最硬核的亮点，值得验证其真实性。

2. **公司选择BIPV、车载等利基市场切入，避开了与晶硅在大规模电站的直接竞争，市场策略有一定合理性。**
   → 如果这些场景的真实需求存在且公司产品性能匹配，有可能在细分市场建立早期优势。


## 为什么不投

1. **实验室稳定性记录与中试线量产数据之间存在巨大鸿沟，BP未提供任何中试线模组稳定性数据或第三方认证。**
   → 这是公司技术优势能否商业化的核心验证点，若无法复现，则技术壁垒不成立。

2. **公司成立仅1年，中试线刚建成，目前零收入、零客户，但BP规划了6条产品线和2025年1500万元收入，扩张故事远超当前业务基础。**
   → 收入预测缺乏真实订单支撑，多条产品线同时推进可能导致资源分散，商业化路径不清晰。

3. **创始团队全为学术背景，无任何光伏产业大规模制造经验，团队能力与业务需求严重不匹配。**
   → 钙钛矿量产的核心挑战是工程化而非实验室技术，缺乏产业经验将极大增加量产失败风险。


## 核心风险

- [P1] **实验室稳定性记录（50000小时）在20MW中试线300mm*300mm模组上无法复现，或数据远低于实验室水平。**
  → 这是公司技术优势能否商业化的核心验证点，若无法复现，则技术壁垒不成立。

- [P1] **所有客户合作均为意向书或战略协议，无任何正式采购合同或付款记录，收入预测缺乏真实支撑。**
  → 如果客户验证无法推进，公司可能长期停留在融资叙事阶段，无法实现商业化收入。

- [P1] **团队全为学术背景，无计划引入产业人才，对大规模制造和供应链管理缺乏认知。**
  → 钙钛矿量产需要深厚的工程化经验，纯学术团队在良率、供应链、客户导入方面存在明显短板。


## 必问问题

1. 你们实验室的50000小时稳定性记录，在20MW中试线的300mm*300mm模组上，实际测到的数据是多少？如果这个数据不能公开，我们怎么相信你们的技术能商业化？
   目的：验证实验室稳定性记录能否在量产模组上复现，这是技术商业化的核心前提。

2. 你们说和某头部新能源车企有战略合作，但BP中写的是‘战略合作协议’。这个协议有没有排他性？对方有没有承诺采购量？如果只是意向，我们怎么判断这不是融资叙事？
   目的：验证客户合作是否真实、有约束力，而非仅用于融资包装。

3. 你们团队全是学术背景，没有一个人有过光伏企业量产经验。如果现在要建200MW产线，你们打算让谁来负责工艺优化和良率控制？如果这个问题回答不了，我们怎么相信你们能商业化？
   目的：验证团队是否意识到自身短板并有计划弥补，这是产业化成功的关键。


## 投资逻辑归因

**核心逻辑**：待验证：硬科技商业化型 vs 制造/产品销售型

**次要逻辑**：项目制交付型, 数据/AI模型驱动型

**风险类型**：硬科技商业化风险, 团队能力风险, 客户验证风险
