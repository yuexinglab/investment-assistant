# 投资决策框架

## 核心判断

**一句话判断**：这是一家以实验室级稳定性世界纪录为核心技术壁垒的钙钛矿材料与界面工程公司，当前最大问题是实验室技术领先与量产商业化能力之间存在巨大鸿沟，且零收入、多市场扩张叙事与有限资源严重不匹配。

**公司本质**：材料与界面工程公司（非组件制造公司）

**决策**：`maybe`（信心度：medium）

**核心原因**：技术方向正确且壁垒高，但商业化验证严重不足，需通过关键缺口验证（模组级稳定性、真实收入、聚焦市场）后方可决定是否推进。


## 值得继续看

1. **稳定性世界纪录（50000小时）是钙钛矿产业化的核心痛点，若能在模组级复现，将形成显著技术壁垒。**
   → 这是公司区别于其他钙钛矿玩家的根本，也是说服客户和投资者的最强卖点，决定了技术护城河的深度。

2. **公司选择从智能项圈、室内光伏等利基市场切入，避免与晶硅正面竞争，若验证成功可快速建立商业化案例。**
   → 利基市场对成本不敏感，对柔性、美观、弱光性能有刚性需求，若客户验证通过，可支撑早期收入和估值。

3. **团队有产业化经验（上海交大实验室搭建）和专业风投背书（海邦创智），具备从实验室走向产业化的基本雏形。**
   → 团队背景和早期投资降低了技术转化和融资风险，增加了项目可信度。


## 为什么不投

1. **实验室稳定性世界纪录（50000小时）基于小面积器件（<1cm²），在300mm*300mm模组上未提供任何第三方认证数据，技术放大风险极高。**
   → 钙钛矿行业的核心挑战是量产工程化，若模组级数据缺失，技术优势可能被高估，投资风险不可控。

2. **公司当前零收入，2025年1500万收入预测缺乏任何客户订单或预付款支撑，商业化路径不清晰。**
   → 收入预测缺乏现实基础，若无法提供合同或预付款，公司仍处于研发/中试阶段，估值应基于技术阶段而非商业化阶段。

3. **公司同时瞄准光伏电站、BIPV、航空航天、智能项圈、室内光伏等多个市场，但团队仅8人，资源极度分散，多线作战可能导致无一突破。**
   → 初创公司资源有限，多市场扩张叙事与单一聚焦资源之间存在严重矛盾，增加失败风险。


## 核心风险

- [P1] **实验室稳定性世界纪录（50000小时）在商业化模组（300mm*300mm）上无法复现，或效率损失超过30%。**
  → 这是公司核心壁垒，若无法在模组级验证，技术优势被高估，投资逻辑崩塌。

- [P1] **公司当前零收入，所有合作仅停留在意向阶段，2025年收入目标缺乏订单支撑，商业化路径存在重大不确定性。**
  → 收入预测缺乏现实基础，若无法提供合同或预付款，公司估值应基于技术阶段，而非商业化阶段。

- [P1] **公司同时进入多个市场（光伏电站、BIPV、航空航天、智能项圈、室内光伏），但团队仅8人，资源分散，可能导致无一突破。**
  → 初创公司资源有限，多线作战增加失败风险，需验证是否有明确聚焦市场。

- [P2] **产能扩张计划（2024年20MW、2025年200MW、2027年GW）激进，资金需求巨大（GW级产线投资约10-20亿元），融资节奏可能跟不上。**
  → 产能扩张需要持续融资，若融资节奏跟不上，规划可能无法实现，公司可能陷入资金链断裂风险。


## 必问问题

1. BP中提到你们拥有2项稳定性世界纪录，持续光照稳定性超过50000小时。这个数据是在什么尺寸的器件上测得的？
   目的：验证稳定性世界纪录的测试条件，判断技术是否可放大到商业化模组。

2. 这个数据在你们刚建成的20MW中试线上，300mm*300mm模组上复现了多少？有没有第三方认证报告？
   目的：确认稳定性记录在模组级别的复现程度，这是判断技术商业化的核心。

3. BP中规划2025年销售收入1500万元，这个收入主要来自哪些产品？是组件销售还是项目交付？目前有没有已经签署的采购合同或预付款？
   目的：验证收入预测是否有真实订单支撑，判断商业化进展。

4. BP中同时瞄准光伏电站、BIPV、航空航天、智能项圈、室内光伏等多个市场，但公司目前只有8个人。你们目前最优先聚焦的是哪个市场？为什么？
   目的：评估多市场扩张策略与有限资源的匹配度，判断公司是否有明确的聚焦路径。

5. 如果现在有一家客户要求采购你们的300mm*300mm模组，并要求提供第三方认证的稳定性数据，你们能给出什么样的保证？如果不能，那这个50000小时的世界纪录，到底是一个实验室成就，还是一个可商业化的产品指标？
   目的：这是红线问题，直接测试公司对技术商业化的真实信心和准备程度。


## 投资逻辑归因

**核心逻辑**：待验证：材料/产品销售型 vs 硬科技商业化型

**次要逻辑**：项目制交付型（利基市场）, 数据/AI模型驱动型（内部工具）

**风险类型**：硬科技工程化风险, 重资产投入风险, 融资节奏风险
