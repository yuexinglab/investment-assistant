# 会前初步判断

## 1. 我现在对公司的定性

**一句话判断**：一家拥有实验室级稳定性世界纪录的钙钛矿技术公司，但当前最大问题是实验室数据能否在量产模组上复现，且无任何收入与真实客户验证，对政府产业基金而言，本地落地、反投和真实客户均未验证。

**公司本质**：实验室技术持有者，尚未验证产业化能力

**决策**：`maybe`（信心度：low）

**核心原因**：项目技术方向有亮点（稳定性世界纪录），但当前阶段（零收入、无客户、中试线刚建成）与政府基金要求的本地落地、反投、真实客户等硬约束严重不匹配，需先验证技术量产可行性和商业化基础，才能判断是否值得推进。


## 2. 为什么还值得继续看（优势 & 相对确定性）

1. **技术壁垒有差异化：宣称的50000小时稳定性世界纪录是钙钛矿行业最核心痛点，若经第三方认证且能在量产中复现，将是颠覆性优势。**
   → 这是项目最核心的潜在价值，也是政府基金偏好技术转移和产业升级的潜在匹配点。

2. **选择高价值小众市场（智能项圈、室内光伏）切入，避免与晶硅正面竞争，策略务实。**
   → 若这些市场能快速产生真实收入，可验证产品商业化能力，为后续本地建厂和反投提供基础。


## 3. 我最担心的几个问题（核心不确定性）

1. **当前无任何产品销售收入，2025年1500万收入规划缺乏现实基础。**
   → 政府基金要求真实客户和收入，零收入项目不符合硬约束，且收入规划可信度低。

2. **与头部车企、玻璃企业的合作仅为意向书或联合开发协议，无任何付款或约束性承诺。**
   → 缺乏真实客户验证，无法判断市场接受度，也不满足政府基金对真实客户的要求。

3. **实验室稳定性数据未获第三方认证，且未在大面积模组上验证，技术产业化风险极高。**
   → 若技术无法量产，本地建厂和反投均无从谈起，与政府基金偏好实体经济和产业落地的目标冲突。


### 主要风险

- [P1] **实验室稳定性世界纪录（50000小时）未获第三方认证，且未在300x300mm模组上复现，技术产业化存在巨大鸿沟。**
  → 这是公司核心壁垒的唯一证据，若无法在量产中复现，技术优势归零，整个投资逻辑崩塌。

- [P1] **公司成立仅1年，中试线刚建成，无任何收入，团队以学术背景为主，缺乏大规模制造和商业化经验。**
  → 从实验室到量产需要大量工程化能力，团队短板可能导致技术无法转化为产品，且资金消耗快。

- [P2] **BP规划多产品线（光伏电站、BIPV、卫星、智能项圈等），但仅有20MW中试线和15人团队，资源极度分散，执行风险高。**
  → 多线扩张叙事掩盖了当前阶段的核心不确定性，可能导致资金和精力分散，无法在任一市场取得突破。


## 4. 和基金匹配度怎么看

**匹配结论**：该项目当前阶段与洪总政府产业基金硬约束（本地落地、反投、真实客户）严重不匹配，需先验证技术量产可行性和商业化基础。

**匹配点**：
- 项目属于新能源/新材料领域，符合政府基金产业升级方向。
- 若技术量产成功，有潜力带动本地光伏产业链和就业。

**不匹配/未验证点**：
- 无任何真实付费客户，不满足硬约束。
- 未提及本地落地意愿或计划，不满足硬约束。
- 未提及反投或地方经济贡献承诺，不满足硬约束。
- 当前仅20MW中试线，产线/产能规划不明确，无法判断本地建厂可行性。

**下一轮必须验证**：
- 下一轮必须验证：公司是否愿意在本地建厂或设立子公司？
- 下一轮必须验证：公司是否接受反投或地方经济贡献要求？
- 下一轮必须验证：是否有真实付费客户或至少已签约的采购合同？


## 5. 下一轮必须验证的问题

1. 你们宣称的50000小时稳定性，是基于实验室小面积器件的数据，还是已经在大面积模组上验证过？如果只是实验室数据，你们凭什么认为它能直接代表产品的户外寿命？
   目的：验证技术壁垒是否具备产业化基础，判断技术风险。

2. 公司成立到现在，有没有产生过任何产品销售收入？哪怕是小批量的测试订单？
   目的：验证公司是否已从实验室阶段进入商业化阶段，判断收入规划可信度。

3. BP中提到的与头部新能源车企和玻璃企业的合作，目前是否已经产生了任何形式的付款或约束性承诺？如果没有，这些合作本质上是不是只是你们单方面的意向表达？
   目的：验证客户合作深度，判断市场验证是否真实。

4. 如果考虑在本地落地，你们是否有意愿在本地建厂或设立子公司？是否接受反投或地方经济贡献要求？
   目的：验证项目与政府基金硬约束（本地落地、反投）的匹配度。
